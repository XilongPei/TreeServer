一般联网问题

    本节解释了几乎所有 Web 节点的基本传输控制协议/网际协议 (TCP/IP) 联网要求，尤其是带多 Web 服务器的节点。针对 Internet 或 Intranet 发布的问题。

路由器和安全设备

    TCP/IP 是可路由选择的协议，意味着每条信息（报文）具有一个指定的路由地址。专用的路由器连接两个网络并在其间路由选择报文。路由器为一个网络上的每个报文检查目的地，如果目的地在路由器的其它网络上，它将报文路由选择到其目的地。

    可将路由器配置为只允许网络间的某些报文，称为报文过滤的进程。报文过滤可用于防止用户查看或连接本地计算机和资源。

    如果具有 TCP/IP 网络，则在网络中可能已有路由器。通常网际服务供应程序 (ISP) 在 Internet 和 Web服务器之间安装路由器。可使用此路由器过滤进入和发出的报文。关于配置路由器或类似的安全设备的信息，请参阅 ISP 或路由器文档。
