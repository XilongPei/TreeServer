读数据字典

　

DWORD tsGetDictInfo( HANDLE hConnect, LPCSTR szGroup,

LPCSTR szLabel, LPCSTR szKey, LPSTR *szCont,

int dwContSize );

　

功能：

请求TreeSVR进行数据字典读。

数据字典结构要略：

[组，GROUP]

#标号，LABEL

关键字，KEY= 数据字典内容

　

参数：

HANDLE hConnect 连接句柄

LPCSTR szGroup 组

LPCSTR szLabel 标号

LPCSTR szKey 关键字

LPSTR *szCont 传回内容的缓冲区

int dwContSize 内容缓冲区大小

　

返回：

出错（可能是没有该关键字）返回0

成功，结果大小
