在Internet上发布信息

    对于到达您的节点世界，您必须有一个 Internet 连接。到 Internet 的连接通常从 ISP 租赁。除了提供物理 Internet 连接和 IP 地址（以及适当的子网掩码），您的 ISP 可提供许多 Internet 服务，比如域名注册、路由器和 DNS 服务。

如何选择正确的 Internet 连接

到 Internet 的连接将通过网卡或其它网络设备，比如调制解调器或综合业务数字网卡(ISDN) 。Internet带宽是以每秒的位数 (bps) 来测量的。

服务器配置和 Internet 带宽决定了数据到达计算机的速度以及可同时服务请求的数量。随着从 Internet连接获取数据的计算机数量增加，除非有足够的带宽，否则将发生延迟或失败。

租用 Internet 连接时，通过 ISP 将网络电缆安装到您的节点。租用的连接在美国境内其速度范围从64,000bps(DDN，数字专线) 到 45,000,000bps(含 T3 连接)，拨号 ISDN 线可提供的最大速度为128,000bps

Internet 连接类型

表10.1中描述的连接类型表示北美和日本境内完全 Internet 连接服务的典型级别。其它国家的 Internet 服务供应商提供的 Internet 服务可能有明显的区别。可以根据硬件性质、节点中可用的内容及其它变量来进一步观察区别。

小负载服务器可用DDN或 ISDN。含中等通信量的服务器可拥有 T1 线或部分已安装的 T1 线。期望大 Internet 通信量的大型商业为处理几千个用户可能需要部分或多条 T1 线甚至 T3 服务。

可向 Internet 连接调制解调器，但它典型地用于单客户浏览，而不推荐用于服务器。使用电话线与调制解调器向 Internet 连接，同时只能服务 2 或 3 个用户（调制解调器连接可能由较少数量的潜在用户用于纯文本的 Internet 服务）。调制解调器连接通常叫做“慢连接”，因为数据以调制解调器的速率传输，典型地在 9,600 到 56,000 bps 之间，远低于 Internet 服务的有效操作。

　

IP 地址和 DNS

Internet 是世界范围内的个人传输控制协议/网际协议 (TCP/IP) 网络的集合。网络上的每台计算机具有唯一地址（IP 地址）。信息以数据报文的形式在 Internet 上传送。每个报文都寻址到特定计算机的 IP 地址，比如 10.212.57.189。因为难以使用和记忆 IP 地址，创建了域名系统 (DNS) 以匹配特定 IP 地址。例如对于 10.189.54.1，给其一个容易识别的域名，比如 microsoft.com。当用户用域名浏览 Internet 时，浏览器首先与 DNS 服务器联系以便识别 IP 地址的域名，然后用此地址与计算机联系。

　

其它 Internet 客户服务

ISP 必须提供连接，一个或多个 IP 地址（以及适当的子网掩码）和至少一个 DNS 服务器的 IP 地址。Internet 服务供应商经常提供附加客户服务。使用这些服务将需要附加软件。

Mail services 用于交换电子邮件，简单邮件传输协议 (SMTP) 和公用邮局协议（POP3）用于 Internet 邮件。

News services 给予您对网络新闻传输协议 (NNTP) 服务器的访问。使用新闻阅读程序，可阅读投递在几千个可用新闻组中的消息。Usenet 是最通用的公共新闻服务之一。
