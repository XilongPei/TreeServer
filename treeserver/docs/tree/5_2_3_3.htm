带参数的立即方式引用ASQL

　

DWORD tsCallAsqlSvrMVar(HANDLE hConnect,LPCSTR szAsqlScript, LPSysVarOFunType lpXexpVar, DWORD dwXexpVarNum, LPSTR szInstResult，DWORD dwInstResultSize，LPVOID lpRsutMem, LPDWORD lpdwLen);

　

功能：

请求TreeSVR给予ASQL服务，同时将符号表传给TreeSVR，TreeSVR在运算时可以引用这些符号，同时也可以将结果放在这些符号中，在ASQL事务完成后，TreeSVR负责将符号表传回来，放在原来的内存空间，API的调用者就可以通过这些符号表获得一些运算结果了。

　

参数：

typedef struct {

short type; //变量类型

char VarOFunName[32]; //变量名

unsigned char values[32]; //值空间

short length; //长度，最大值为32

} SysVarOFunType *LPSysVarOFunType;

type的可取值为：

整型：INT，1054

长整：LONG，1055

浮点：FLOAT，1056

字符：CHAR，1053

字符串：STRING，1051

日期：DATE，1052

所有变量的值都在values中，字符串最长31字符，另有一尾0，具体存法参见ASQL文档。

lpXexpVar：参数缓冲区

dwXexpVarNum：参数数量

szAsqlScript：脚本字符串，以尾0结束

lpRsutMem：存放长大结果的缓冲区

lpdwLen：缓冲区大小

szInstResult:立即结果。为一指向字符缓冲区指针。

sprintf(szInstResult, "ERR:%d INF:%s\n%s\nXEXP:%s", \

ERR:ASQL错误号 \

ASQL错误信息, \

ASQL错误附属信息, \

表达式错误信息 {表达式错误附属信息}

);

dwBufSize：立即结果缓冲区大小

　

返回：

1:完整结果

2：不完整

3：不成功

tsGetLastError()可获得出错信息。
