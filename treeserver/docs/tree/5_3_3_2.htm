用户级帐号操作协议

　

msgType = 'U'

　

修改密码

lp = 0x4000 len =旧密码长度 + 1 + 新密码长度 + 1 包数据为旧密码\x0新密码\x0

返回：lp = 0 成功；

否则 lp 为出错号，len 为出错信息长，数据为出错信息

　

　

在 TreeSVR 的用户帐号数据库中，包含一个内置的帐号，对于用户是不可见的，该帐号的标识为 "#T_USER"，存储的是用户的模板信息及帐号的规则。 所谓的模板即用户帐号参数的默认值，描述如下：

　

szUser* "#T_USER"

dwUserId* 目前可用的最小用户 ID

系统内部保留，与应用程序无关

szPasswd* ""

szHomeDir+ 默认为 "%USER_HOME%\%USER_NAME%"

%USER_HOME% 表示代表 TreeSVR 用户的根目录的变量

%USER_NAME% 表示代表当前用户名的变量

在实际运行时是用 TreeSVR 用户的根目录和当前用户名

代换的。

szLogonComputer* ""

　

bLogonHours+ 用户默认的登录时间控制

　

szAccessMask+ 用户默认的权限掩码

　

szDescription* ""

　

cLogonFail- 用户被锁定的最大登录失败计数

cLocked- 是否根据登录失败计数 cLogonFail 的值锁定帐号

cLogonFail 与 cLocked 两个标志是相互作用的，当某用

户帐号的登录失败计数达到该 cLogonFail 值时，检查

cLocked 是否为 1，若为 1 则锁定帐号，若为 0 不产生

用户帐号的锁定。
