用户引擎的设置

　

用户引擎集中的存放在 <TreeServerInstallDir>/Engine/ 目录下，配置文件在<TreeServerInstallDir>/Config/TEngine.ini 中。

/**********************************/

/*ASQL为内置引擎*/

[Transaction Engine]

iSQL = Informix ODBC function engine

File = File Service engine

Hello = test

David =

[iSQL]

Engine = c:\Program Files\TreeServer\Engine\isql.dll

Command = isql

EntryFunction = iSQLEntry

Capability = m-s

EnableTimer = 0

EnableIdle = 0

TimeOut = 100000

Enable = 1

[FILE]

Engine = c:\Program Files\TreeServer\Engine\filesvc.dll

Command = File

EntryFunction = MyExportFunction

Capability = m--

EnableTimer = 0

EnableIdle = 0

TimeOut = 60000

Enable = 1

[Hello]

Engine = c:\Program Files\TreeServer\Engine\Hello.dll

Command = Hello

EntryFunction = MyExportFunction

Capability = m-s

EnableTimer = 0

EnableIdle = 0

TimeOut = 60000

Enable = 1

[David]

Engine = c:\Program Files\TreeServer\Engine\David.dll

Command = David

EntryFunction = MyExportFunction

Capability = m-s

EnableTimer = 0

EnableIdle = 0

TimeOut = 60000

Enable = 1

说明：

Capability = mes

大小写不区分，M表示多线程的，E表示是否独占，S是否自身具备调度能力。其否定用减号表示。
