TreeServer 的客户访问

TreeServer 是以命名管道的方式提供用户接口的，现在支持 Windows 95 或 Windows NT 做为客户端，可以支持 Microsoft 操作系统所支持的所有协议。

当用户在 Windows 95/NT 上存取 TreeServer 服务器时，用户须持待访问服务器上的隶属于 "TreeServer Users" 用户组的帐号在客户机上注册登录，才能具备访问 TreeServer 服务网络接口的权限。

当用户要通过 Web 服务器来访问 TreeServer 服务器时，须对 Web 服务器的匿名帐号进行设定，该 Web 服务器的匿名帐号应使用一个同 TreeServer 服务器上的隶属于 "TreeServer Users" 用户组的某一帐号完全相同的用户名及密码，该帐号信息同时在域用户管理器中进行修改。
