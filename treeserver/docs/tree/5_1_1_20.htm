ASQL语法语句，及变量表

请求包格式：

ASQL脚本。

len = 本包信息长度

lp = 总长度

应答包格式：

由三部分组成：包头

packetType = 'R';

msgType = 'D';

len = 信息长度，szInstResult，立即结果长度

lp = 0，成功，其它各种原因

立即结果，最长为一个包长。

执行结果：多个包，由三部分组成：包头

packetType = 'R';

msgType = 'X'; //服务器下传数据

len = 本包信息长度

lp = 0

立即结果，最长为一个包长。

符号表交换信息包：

packetType = 'R';

msgType = 'V'; //服务器下传数据

len = 本包信息长度

lp = 变量个数

　

TreeSVR客户端与服务器端交换的变量若为字符串，则最长只能为31字节（有一尾0），在从客户向服务器端传此信息时，要校验此信息。
