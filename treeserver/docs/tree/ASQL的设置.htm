ASQL的设置

　

ASQL 的数据字典存放在 <TreeServerInstallDir>/Config/asqlwmt.cfg 文件中。示例内容如下：

//ASQL事务服务引擎配置文件

//版权所有: (c)上海中铁高新软件中心 1997，1998

//

//本配置文件每行最长 254 ASCII字符

[MACRO]

PEI=YINZHU

[SYSTEM]

//CODE=e:\wmtasql\code\code.cod

CODE=g:\progra~1\treese~1\code\code.cod

LOGFILE=C:\Progra~1\TreeSe~1\LOG\TreeSvr.Log

LASTTID = 34

[FIELDCTL]

PATH=e:\wmtasql\fieldctl.dbf

[POLICY]

#OPENDBF

//NUM = 2

//F1 = LZ0.DBF

//F2 = LZ1.DBF

[DATABASE]

#Stores

PATH = e:\wmtasql\

#LZ

PATH=C:\Progra~1\TreeSe~1\Data\LZ\

#TG

PATH=C:\Progra~1\TreeSe~1\Data\tg\

#Hospital

PATH = D:\Hospital\

#INSURE

PATH = C:\Progra~1\TreeSe~1\Data\Insure\

[DBROOT]

#People

KEEP_PHYSICAL_ORDER = 1

DPACK_AUTO=0

TID=32

NDX1=People

KEY1=h_i

#Medicine

KEEP_PHYSICAL_ORDER = 1

DPACK_AUTO=0

TID=33

NDX1=code

KEY1=code

NDX2=spell

KEY2=spell

[PATH]

#DBROOT

PATH = C:\Progra~1\TreeSe~1\Data\

[DBROOT]

#LZ0

NDX1=

KEY1=

　

数据字典段描述：

1.[MACRO]

宏定义

2. [SYSTEM]

系统

代码：

CODE=g:\progra~1\treese~1\code\code.cod

日志文件名

LOGFILE=C:\Progra~1\TreeSe~1\LOG\TreeSvr.Log

最大表标识号（内部用）

LASTTID =

3. [FIELDCTL]

域定义表文件名

PATH=e:\wmtasql\fieldctl.dbf

4. [POLICY]

优化

预先打开文件

#OPENDBF

//NUM = 2

//F1 = LZ0.DBF

//F2 = LZ1.DBF

5. [DATABASE]

数据库定义

#Stores //库名

PATH = e:\wmtasql\ //文件目录

　

6.[DBROOT]

缺省数据库定义

#People //表名

KEEP_PHYSICAL_ORDER = 1 //保持物理序

DPACK_AUTO=0 //自动紧缩

TID=32 //表标识号

NDX1=People //索引名1

KEY1=h_I //索引关键字2
