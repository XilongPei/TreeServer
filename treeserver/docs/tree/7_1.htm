内部Web方式信息系统配置

1硬件

建立一个WEB方式信息系统需下列硬件

(1)PC SERVER

64M内存以上，4GB以上硬盘，至少一个Pentium级CPU。

(2)局域网络

所有本地机器应连成一个局域网，以利信息共享，连一个局域网一般需下列设备：

每台机器一块网卡、集线器HUB一台、双绞线若干米、RJ45插头每台机器二个。

如果系统运行在比较大的范围，还需下列设备：

(3)路由器

如cisco 19000, 3com, Bay都可，价格约二万元。

(4)调制解调器

每条电话线连入需一台调制解调器。

　

2软件

Windows NT 4.0以上操作系统

TreeSVR事务代理服务器

浏览器，如《Netscape通讯》。
