TreeServer安装

　

TreeServer 是以 CDROM 的形式发行的，安装需要将 CDROM 介质放入 CDROM 驱动器，假设其驱动器盘符为 X:，运行

　　X:\TreeSvr\Setup.exe

安装程序将 TreeServer 系统及相关文件复制到指定的位置，但并没有自动的配置其运行所需的环境，待安装完成后须用户手工配置。（目前安装程序还有待完善）

TreeSVR 客户端无特别的安装要求，只需运行 TreeSVR 客户应用的系统可以登录到运行TreeSVR 的 Windows NT 服务器上即可。这取决于 TreeSVR 服务器与客户之间的通信是通过管道实现的，客户对管道的访问需通过 Windows NT 的安全审核。

　　在安装的过程中须输入的信息包含用户信息、安装目标路径、程序组的名称等，用户可根据实际情况和需要填写。

　　安装结束后既可对系统进行配置，详细信息参见 TreeServer 的系统设置，之后重新启动计算机，然后通过 TreeServer 服务的管理界面来启动运行该服务。
