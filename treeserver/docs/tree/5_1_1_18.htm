ASQL语法文件方式请求ASQL服务

　

msgType = 'B'

ASQL语法文件方式请求ASQL服务。

　

请求包格式：

256字节：ASQL脚本文件名，该脚本必须已经在服务器上。

应答包格式：

立即响应结果：一个包，由三部分组成：包头

packetType = 'R';

msgType = 'D';

len = 信息长度，szInstResult，立即结果长度

lp = 0，成功，其它各种原因

立即结果，最长为一个包长。

执行结果：多个包，由三部分组成：包头

packetType = 'R';

msgType = 'X'; //服务器下传数据

len = 本包信息长度

lp = 0

立即结果，最长为一个包长。
