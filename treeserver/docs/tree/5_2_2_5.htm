取记录

　

DWORD tsTableGetRec(HANDLE hConnect,DWORD dwTableId,DWORD dwRecNo,LPSTR szRecBuf, DWORD dwBufLen,LPRECPROPS lpRecProps)

　

参数：

hConnect:标识本连接。

dwTableId：表标识。

dwRecNo：记录号

szRecBuf记录缓冲区，记录缓冲区必须足够容纳记录。

dwBufLen:缓冲区长度。

lpRecProps:记录状态

typedef struct tagRECProps {

long iSeqNum;

long iPhyRecNum;

short iRecStatus;

short bSeqNumChanged;

short bDeleteFlag;

delete supported only }

} RECPROPS;

　

返回：

出错号,0表成功。
