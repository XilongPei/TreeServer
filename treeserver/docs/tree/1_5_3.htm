TreeSvr.EXE命令行

　

可以通过TreeSVR调度程序来起动整个TreeSVR服务。

1.装入服务

TreeSvr -install to install the service

2.撤消服务

TreeSvr -remove to remove the service

3.开始服务

TreeSvr -start to start the service

4.停止服务

TreeSvr -stop to stop the service

5.服务状态

TreeSvr -status to query the service status

6.调试运行

TreeSvr -debug <params> to run as a console app for debugging

7.创建用户数据库

TreeSvr -UserDB Create the default user account database
