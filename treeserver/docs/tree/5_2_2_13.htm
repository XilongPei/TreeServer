取表的域信息

　

DWORD tsTableGetFieldInfo(HANDLE hConnect,DWORD dwTableId,LPDIO_FIELD szFieldInfo, DWORD dwFieldLen)

　

参数：

typedef struct {

unsigned char field[32];

unsigned char fieldtype;

unsigned char fieldlen;

unsigned char fielddec;

long fieldstart;

} DIO_FIELD, *LPDIO_FIELD;

数组的最后一个元素的field为""

hConnect:标识本连接。

dwTableId：表标识。

szFieldInfo：域信息缓冲区

dwFieldLen:缓冲区长

　

返回：

域数量
