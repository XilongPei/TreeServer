什么是Intranet

    Intranet指的是任何未连接到Internet，但是使用 Internet 通讯标准和工具，在私人网络上为用户提供信息的任何 TCP/IP 网络。例如，公司可安装 Web 服务器，只有员工可用于发布公司业务通讯、销售图表及其它的公共文档。员工使用 Web 浏览器访问信息。 用与 Internet 上相同的特性和服务配置 Web 服务器以提供 Intranet，例如，服务超文本页面（它可包含文本、超级链接、图象及声音）、响应 Web 客户对信息的请求及访问数据库。在本手册中，无论这些发布服务是在 Intranet 上还是 Internet 上运行，都将其描述为“Internet 服务”。
