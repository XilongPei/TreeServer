客户端

  • WindowsNT

同服务器一样，需要添加NETBIOS服务和远程访问服务以及点对点通道协议，只不过在远程访问服务的配置按钮中需将解调器端口设置为“只能拨出”或“拨出和接收”，在其网络配置面板中将拨出协议设为TCP/IP即可。

  • Windows95

在控制面板中正确添加调制解调器，在网络配置中添加Microsoft网络客户、拨号网络适配器、TCP/IP协议，同局域网一样，正确配置TCP/IP协议的IP地址，DNS服务器，和网关等项。然后在我的电脑中的拨号网络选项新建一个拨号连接，在其连接属性中最主要配置好服务器类型的选项：将拨号服务器类型设置为“PPP Windows 95, Windows NT 3.5, Internet”；高级选项多选框选择登录到网络，启用软件压缩；允许的网络协议只选择TCP/IP，并将TCP/IP设置与控制面板中的网络设置保持一致，其他设置保持默认即可。
