什么是Internet

　

    Internet 是使用公共语言进行通讯的全球计算机网络。它类似于国际电话系统无人拥有或控制整个系统，但是以大型网络的工作方式连接。

World Wide Web（WWW 或简称 Web）为用户在 Internet 上查看文档提供了一个图形化的、易于进入的界面。这些文档及其之间的链接，组成信息“网”。 Web 上的文件或页面是互相联系的。通过单击特定的文本或图形连接其它页面，称为超级链接。页面可以包含文本、图象、动画、声音等所有的东西。可将这些页面置于世界任何地方的计算机上。当您连接 Web 时，即可访问全世界范围内的信息。 超级链接是 Web 地址嵌入其中的字或图形。单击超级链接可跳到特定 Web 节点中特定的页面。可以容易地标识超级链接。超级链接文本与 Web 页面上的其余文本通常颜色不同，超级链接图形通常有带颜色的边界。

每个 Web 页面，包括 Web 节点的主页，具有唯一的地址称为统一资源定位符(URL)，例如：

http://www.crsc.shtdu.edu.cn/home.htm。

URL 指定存储页面的计算机名以及到此页面的确切路径。
