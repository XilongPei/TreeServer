历史资料说明信息库

  历史资料数据集为 HISTRSUT.DBF，用来存放历年的统计结果资料，即每次根据历史资料分析条件库所采集的数据。

其结构如下：

COND_CODE 分析条件代码(与 HISTCOND.DBF中的对应域的定

义相同)，宽度为6，字符型

STAT_ID 数据时间点说明(与 HISTSREM.DBF中的对应域的

　

定义相同)，宽度为6，数字型

RESULT 历史资料数据，宽度为32，字符型

RESULT_TYPE 分析结果数据类型，宽度为1，字符型

　

RESULT_TYPE= N 整型结果

RESULT_TYPE= F 浮点型结果

RESULT_TYPE= C 字符串型结果
