Ftp 常用命令参考

Ftp: help

显示 Ftp 命令的解释。
help [command]

参数
   command
   指定要解释的命令名称。如果未指定 command ，ftp 将显示包含所有命令的列表。

Ftp: dir

显示远程目录的文件及子目录列表。
dir [remote-directory] [local-file]

参数
   remote-directory
   指定要查看列表的目录。如果没有指定目录，则使用远程计算机上的当前工作目录。
   local-file
   指定保存列表的本地文件。如果未指定文件，则在屏幕上显示输出。

Ftp: binary

将文件传输类型设置为二进制。
binary

注意
   FTP 支持两种类型文件传输。ASCII 和二进制映象。传输可执行文件时，应该使用二进制类型。在二进制模式中，文件是逐字节传送的。请参阅 Ftp_ascii。

Ftp: ascii

默认情况下，将文件传输类型设置为 ASCII。
ascii

注意
   FTP 支持两种文件传输类型：ASCII 和二进制映象。传输文本文件时，应该使用ASCII。请参阅 binary。在 ASCII 模式中，执行字符与网络标准字符集之间的转换。例如，要根据目标操作系统的需要转换行结束符。

Ftp: get

使用当前文件传输类型，将远程文件复制到本地计算机。
get remote-file [local-file]

参数
   remote-file
   指定要复制的远程文件。
   local-file
   指定本地计算机上的文件名。如果未指定，则该文件使用remote-file名称。

Ftp: put

使用当前文件传输类型将本地文件复制到远程计算机。
put local-file [remote-file]

参数
   local-file
   指定要复制的本地文件.
   remote-file
   指定复制到远程计算机上的文件名。如果没有指定，则使用与 local-file 相同的文件名。

Ftp: mget

使用当前文件传输类型将多个远程文件复制到本地计算机。
mget remote-files [ ...]

参数
   remote-files
   指定要复制到本地计算机的远程文件。

Ftp: mput

使用当前文件传输类型，将本地文件复制到远程计算机。
mput local-files [ ...]

参数
   local-files
   指定要复制到远程计算机的本地文件。

Ftp: pwd

显示远程计算机上的当前目录。
pwd

Ftp: mkdir

创建远程目录。
mkdir directory

参数
   directory
   指定新远程目录的名称。

Ftp: delete

删除远程计算机文件。
delete remote-file

参数
   remote-file
   指定要删除的文件。

Ftp: rmdir

删除远程目录。
rmdir directory

参数
   directory
   指定要删除的远程目录名。

Ftp: cd

更改远程计算机上的工作目录。
cd remote-directory

参数
   remote-directory
   指定要进入的远程计算机目录。

Ftp: rename

更名远程文件。
rename filename newfilename

参数
   filename
   指定要更名的文件。
   newfilename
   指定新文件名。

Ftp: bye

结束与远程计算机的 FTP 会话，并退出 ftp。
