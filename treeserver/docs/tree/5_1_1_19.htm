ASQL脚本方式请求ASQL服务

请求包格式（QA）：

ASQL脚本。

len = 本包信息长度

lp = 总长度

leftPacket可能不为0，后面还有包，即本请求是由多个包组成的。

endPacket可能不为0，后面可能还要传别的包，如符号表。

符号表交换信息包定义：

packetType = 'R';

msgType = 'V'; //服务器下传数据

len = 本包信息长度

lp = 变量个数

应答包格式：

由三部分组成：包头

packetType = 'R';

msgType = 'D';

len = 信息长度，szInstResult，立即结果长度

lp = 0，成功，其它各种原因

立即结果，最长为一个包长。

执行结果：多个包，由三部分组成：包头

packetType = 'R';

msgType = 'X'; //服务器下传数据

len = 本包信息长度

lp = 0

立即结果，最长为一个包长。
