关键字说明：

PAR par(价值相等的，来源于容错系统设计语言)
TB Transaction Block, 事务块
Transaction Name 事务名称
Transaction Type 事务类型，TreeSVR根据事务类型决定引用哪个服务器来完成

其它说明：

一个PARDO段称为一个并发事务块（par transaction block）。
一个PARDO段可以包含多个可以并行执行的事务，这些事务完成各自的任务。
执行PARDO段内描述的事务的用户身份是本ASQL角本的用户身份。
